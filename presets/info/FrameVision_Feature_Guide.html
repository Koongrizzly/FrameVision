FrameVision — Feature Guide
Local AI media studio • Video, image, music, speech, queue, planning & performance tools

Features

🧪 Upscaler

  • Image and video upscaling with queue support, recents and player integration.
  • Quality + sound options for video; source-FPS friendly workflows.
  • Models: Real-ESRGAN family, RealSR x2/x4, Upscayl, SRMD, UltraSharp, SeedVR2, anime video models, Waifu2x family.
  • Batch tools, format/output controls and result browsing.

🖼️ Image Edit & Reference Images

  • Local image-edit workflows with reference-image guidance for stronger consistency and targeted changes.
  • Useful for repainting, identity consistency, prompt-guided edits and reference-based creative work.
  • Models: Qwen Edit 2511, HiDream, Flux Klein and FireRed 1.1.
  • Great for editing characters, products, scenes and style while keeping key details recognizable.

⏱️ RIFE Interpolation

  • Queue-native frame interpolation and slow motion.
  • Slow-mo 0.15×–1.0× with audio retime; interpolation 1.0×–4.0× for higher FPS.
  • Different profiles and models for up to 4K workflows.
  • Fast result access through recents and player preview.

📝 Describe (AI)

  • Engines: Qwen3-VL (2B Instruct) + BLIP for quick captions.
  • Offline scene understanding, prompt making and OCR-like reading.
  • Useful presets for general descriptions, prompt conversion and troubleshooting.
  • Works with current frame, single files or folders.

💬 LLM Chat

  • Local chat workspace with a GGUF LLM loader powered by llama-server.exe.
  • Useful for brainstorming, writing help, prompt support and quick questions without leaving FrameVision.
  • Exclusive FrameVision actions can create images, videos and music, edit or upscale results, and keep finished files in the chat.
  • Right-click menus and in-chat players make it fast to continue, edit, upscale, delete or review generated media.

🖌️ TXT→IMG

  • Local image generation with classic and newer engines in one workspace.
  • Prompt builder, negatives, size presets, aspect lock and queue support.
  • Fast reruns, player preview, JSON sidecars and batch/seed control.
  • Great as the central image-generation hub inside FrameVision.

🧠 New Image Models

  • Qwen-Image-2512 (GGUF) for realism, strong text rendering and local generation.
  • Ideogram 4 GGUF with Turbo / Default / HQ modes.
  • Lens and Chroma integrated as newer image-generation options.
  • Z-Image Turbo GGUF and FP16 path for fast concepting on many systems.

🧹 Background & Inpaint

  • Dedicated background remover/replace workflow with fast and heavier removal models.
  • Brush editing, magnifier, pan, undo/redo and before/after preview.
  • Local SDXL inpainting for editing masked image areas.
  • Settings such as CPU offload, attention slicing, VAE tiling and size matching.

🧰 Multi Tool

  • Central utility area for video, image, audio and file-management tasks.
  • Includes trim, split/glue, resize/convert, crop, reverse/boomerang, GIF creation and extract-frames workflows.
  • Also covers prompt enhancement, Qwen 3 TTS, Whisper Lab, metadata editing and multi rename/replace.
  • Great for practical day-to-day jobs that do not need a full generation pipeline.

🎞️ Video Timeline Editor

  • Mini OpenShot-style local editor for building videos from clips, images, audio and text/title assets.
  • Multi-track timeline with trim, split, move, copy/paste, duplicate, snap, track mute/solo/lock and multi-selection.
  • Visual transforms, keyframes, fades, transitions, reverse playback, text animations and audio mixing tools.
  • Project JSON save/load, autosave recovery, preview quality controls and final MP4 export with FFmpeg.

🎞️ Video Generation

  • LTX 2.3, Wan 2.2 and HunyuanVideo 1.5 workflows.
  • TXT→Video, IMG→Video, Video→Video, extend and more advanced start/end guidance where supported.
  • Works with FrameVision queue and player-oriented result flow.
  • Designed to pair with VRAM Lab for larger local model runs.

🧠 Planner

  • Turns one idea into a structured video plan with beats, shots and prompts.
  • Character Bible support plus own-storyline and alternative-storyline modes.
  • Can generate clips per shot, narration, music and a final assembled cut.
  • Resume/review workflows through Results & History.

🖼️ Thumbnail / Meme Creator

  • Use the current frame/image or load a new one with big preview.
  • Multiple text layers, outlines, shadows, ALL CAPS, alignment and readability tools.
  • Interactive crop overlay with 16:9 / 9:16 / free framing.
  • Transform sliders for rotate, tilt/skew and arc curvature.

🎬 Music Videoclip Creator

  • Auto-cuts to beat and energy with effects, transitions and drop/chorus styling.
  • Can use timeline overrides for more manual control.
  • Supports beat-synced visuals and newer LTX-oriented workflow expansion.
  • Built to turn generated clips and music into a polished music video quickly.

🎼 Music Creation

  • Ace Step 1.5 music generation with prompt/tags, negatives and optional lyrics.
  • Genre presets, favorites, seed control and result playback.
  • Supports instrumental or lyric-guided workflows for broader music creation.
  • Integrates naturally with Music Videoclip Creator and Planner.

🎙️ Qwen 3 TTS & Whisper

  • Qwen 3 Text-to-Speech: base voice, custom voice, clone voice and voice design.
  • Whisper for subtitles, transcripts and speech-to-text workflows.
  • Useful for narration, planning, lyric/subtitle work and accessibility.
  • Complements the Planner’s story → narration → final video pipeline.

📋 Queue

  • Central processing queue with running, pending, finished and failed management.
  • Rich job rows, ETA/progress, auto-play toggle and internal player integration.
  • Supports many FrameVision tools instead of each tool needing its own mini-queue.
  • Safe cleanup, output resolving and robust file handling.

🎬 Media Player

  • Offline player for video, image and audio with fullscreen, zoom, pan and compare.
  • Built-in music player with playlist and 250+ beat-synced visuals.
  • Player-first workflow helps keep tools connected to the same media view.
  • Good for previewing, comparing and checking generated results directly inside FrameVision.

🗂️ Media Explorer

  • Dedicated browser for generated images, videos, audio and project result folders.
  • Search, sort, preview, rename, delete and favorite media without leaving the app.
  • Useful for keeping large output folders organized after many model runs.
  • Works alongside the player so results can be opened and checked quickly.

🧪 VRAM Lab & DDR Manager

  • Beta memory-management layer for heavy local models.
  • Helps reduce shared-memory spill and makes larger runs more practical.
  • Works toward reusable model-agnostic control for LTX, Wan, Hunyuan and more.
  • Paired with auto profiles and future memory preflight checks.

⚙️ Settings, Downloads & Model Manager

  • Theme controls, output paths, maintenance tools and UI toggles.
  • Optional Downloads menu for installing newer large models only when needed.
  • Auto update checks, logs toggles, font size options and visual/UI preferences.
  • Remove/hide tools help keep the app clean when some models are not needed.

🧾 Results / History

  • Planner-oriented history view for finished, partial and failed jobs.
  • Resume is the key workflow feature for continuing unfinished projects.
  • Colored asset indicators show what stages of a workflow have completed.
  • Open folder, manifest, errors or favorite important jobs.

Overview

🧭 Workspace

FrameVision keeps a left-side media viewer and a right-side tool area so generation, editing, playback and browsing stay connected. The app is built around a practical local workflow instead of many disconnected mini-apps.

Windows CPU / GPU PySide6 Local-first

🔁 Core workflow

  • Open or generate media, preview it in the internal player, then send it into the next tool.
  • Use queue where it makes sense, or run directly in tools that are meant to be immediate.
  • Let Planner handle bigger idea → final-video pipelines.

🧩 Optional downloads

  • The app no longer depends on an old “full CPU install” idea.
  • Large models are installed from the Optional Downloads menu when needed.
  • This keeps the base app lighter and gives users more control over disk space.

Video Timeline Editor — more info

Local timeline editing • Multi-track projects • Text, transitions, keyframes and FFmpeg export
  • Build a normal edit by adding videos, images, audio and reusable Text assets to a multi-track timeline.
  • Drag clips to move them, drag edges to trim, split at the playhead, duplicate/copy/paste sections, and use Shift-click multi-selection for grouped edits.
  • Visual clips can be moved, resized, rotated, mirrored/flipped, faded and animated with transform keyframes for zooms, pans, picture-in-picture and motion effects.
  • The Text/Title engine supports reusable title assets, clip-specific placement, fades, typewriter/reveal, slide/zoom, scroll credits, rotation and pop/bounce style animations.
  • Transitions are made from overlapping visual clips and PNG mask files; optimized grayscale masks keep live preview smoother.
  • Project save/load keeps the edit structure, while autosave can recover unsaved work. Export creates the final MP4, so short test exports are the best way to judge heavy transitions or stacked effects.

Queue — more info

Central job handling • Rich rows • Safer file/result logic
  • Supports running, pending, finished and failed states, plus cleanup and recovery actions.
  • Rows can show titles, subtitles, progress, ETA and output-related actions.
  • Internal player is preferred for quick result viewing; OS fallback can still be used when needed.

Media Player — more info

Playback, preview and comparison
  • The internal player supports video, audio and images with fullscreen, zoom/pan and compare-friendly behavior.
  • The built-in music player includes playlist support and 250+ beat-synced visuals.
  • Player-first workflow keeps many tools connected to the same preview area, so users can check outputs without constantly opening external apps.

Media Explorer — more info

Search, sort, preview and manage generated media
  • Media Explorer is a separate browser for generated output folders and media libraries.
  • It helps users search, sort, preview, rename, delete and favorite results folders/files.
  • This is especially useful when many model runs create a large number of images, videos or audio files.
  • It works alongside the internal player, so selected results can be opened and checked quickly.

VRAM Lab & DDR Manager — more info

Model-agnostic memory-management direction for heavy local AI workflows
  • VRAM Lab is a reusable backend direction rather than a single-model hack. It aims to help LTX, Wan, Hunyuan and future large models stay more manageable under local hardware limits.
  • Current focus includes reducing shared-memory spill, using safer memory profiles and improving practical large-model runs.
  • DDR Manager is the paired idea for RAM/pagefile/commit preflight checks and safer heavy-run decisions.
  • These systems are especially important for users running advanced video models on limited or mid-range VRAM.

Settings, Downloads & Model Manager — more info

Customization, maintenance and model control
  • Settings cover paths, themes, UI behavior, logs, font size and other quality-of-life controls.
  • The old “full CPU install” style wording can be dropped in favor of a clearer Optional Downloads model.
  • Auto update checking and restart prompts help keep the app current.
  • Remove/hide tools can reduce clutter and help users manage installed model environments and optional components more safely.

Results / History — more info

Planner-focused project tracking • Resume-heavy workflow
  • Shows past jobs with status like Failed, Partial and Done.
  • Colored asset indicators help users understand which workflow parts completed for each job.
  • Actions can include opening the folder, manifest or errors, resuming a run, removing the folder or marking favorites.
  • Resume is the most important action here because it turns FrameVision into a more serious long-project environment.
