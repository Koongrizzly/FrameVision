All-in-one Video & Photo Upscaler/Editor • Tabs, Workflow & Install

Features

🧪 Upscaler

  • Real-ESRGAN family & friends. recent results
  • Quality + sound options for video
  • Batch, single; queue integration.
  • Models: realesrgan-x4plus, realesrgan-x4plus-anime; realSR x2 x4, Upscayl; realesr-animevideov3-x2/x3/x4; Waifu2x: models-cunet, models-upconv_7_photo, models-upconv_7_anime_style_art_rgb.

🖼️ Thumbnail / Meme Creator

  • Use current image/frame or open image; live preview.
  • Multiple text layers; fonts, outline width, shadow, ALL CAPS, align.
  • Transform sliders: rotate, tilt/skew, arc curvature.
  • Interactive crop overlay (16:9 / 9:16 / free).
  • Export PNG/JPG; EXIF keep/remove; big preview.

⏱️ RIFE Interpolation

  • One‑click frame interpolation, batch, queue‑native.
  • Slow‑mo 0.15×–1.0× (audio retime); 1.0×–4.0× interpolation for higher FPS.
  • Different models for up to 4k
  • Last results.
  • Speed profiles with one very fast profile (gpu only) ”.

📝 Describe (AI)

  • Engine: Qwen3‑VL (2B Instruct) + BLIP (quick captions).
  • Offline describing: rich captions, OCR‑ish reading, scene understanding.
  • Presets for different styles (general, prompt‑maker, troubleshooting).

🖌️ TXT→IMG

  • SDXL + SD1.5 + Z‑Image (Turbo GGUF) — fully local image generation.
  • Quick presets + prompt builder; optional negative prompt.
  • Size presets + manual W×H; lock aspect ratio for clean framing.
  • Queue‑native runs with player preview and fast reruns.
  • Sampler/steps/seed controls when you want to fine‑tune.

🧰 Tools

  • Speed, Resize, GIF export, Frame extract, Trim, Crop, Quality & Convert.
  • Reverse Video (boomerang), Compare, Multi‑Rename, Add Audio.
  • Batch tools for folders; queue‑friendly.

🧹 Background remover (BETA)

  • Use currently loaded or add new image
  • 2 models for removing backgrounds (1 fast and one heavy model) + sd15 loader for inpainting. (default sd15 inpainter included)
  • Select background : transparent/color/blur/image
  • Preview panel with brush, magnifier + pan undo and redo
  • Currently in beta for testing.

⚙️ Settings

  • Paths: models root, bin (FFmpeg, RIFE). Output dirs.
  • Performance: Prefer CPU, DirectML toggle, Precision (FP16).
  • Downloads: set Hugging Face cache (./.hf_cache).
  • Theme: live theme switcher (Day / Evening / Night / Cyberpunk / Neon / Ocean / Solarized Light / CRT) and many more.
  • Temperature units: switch between Celsius (°C) and Fahrenheit (°F).
  • Maintenance: clear program cache, dump QSettings, open log folder.
  • Session: restart the app safely (saves config, reloads UI).
  • Boot options: random intro on startup; (random) overlay animations with preview. easter egg inside the startup intro (click 4x)
  • UI optimze : reorder, use emoji pack (with tab text on/off), use banner (with color on/off), swap ui,....

📋 Queue

  • Recover Running → Pending.
  • Mark Running → Failed; Clear Finished/Failed.
  • Groups: Running, Pending, Finished (collapsible).
  • Per-job title + subtitle with model, size, timings.
  • Left status strip: blue running, green done, red failed, yellow pending.
  • Progress bar with ETA; indeterminate when unknown.
  • Auto‑play newest result (toggle; remembers setting).
  • Play via internal player first; OS fallback.
  • Open output folder, Play, Delete buttons + context menu.
  • Recents integration; sets current video/path when available.
  • Live refresh while running; palette/style change aware.

🎬 Media Player

  • Offline player for video, images, and audio.
  • Fullscreen with auto‑hide controls; zoom 50x + pan (pause your video if you want to zoom in 50x to avoid stuttering) ; quick info HUD and extra 'more info' button.
  • Quick Compare/wipe loader for A/B (image or video) + repeat button for videos.

🗃️ Media Explorer

  • Browse your results folders (search, sort, reorder (most) tabs, add to/show only favorites).
  • Play/preview, rename, delete, and open‑in‑folder.
  • Scans to load extra json info. (can take some time on huge folders with video files)

🎞️ Video Creation

  • WAN 2.2 + HunyuanVideo 1.5 engines.
  • TXT→Video, IMG→Video, Video→Video, and Extend workflows.
  • Queue‑native runs with VRAM/offload options where supported.

🎵 Music Player

🎬 Music Videoclip Creator

🤖 Ask (Framie)

  • Offline helper for workflows, troubleshooting, and “how do I…”.
  • Can reference your current media/frame + settings context.
  • Modes like Coach / Power User / ELI5 / Bug‑hunt.

📊 HUD & Stats

  • Header HUD shows CPU/GPU/VRAM + internet traffic (when active).

🗂️ File menu — Multi‑convert & Save

Editor — timelines & shortcuts

CapCut/OpenShot‑style editing • Snapping • Ripple • Markers
  • Media panel: thumbnails with quick preview (double‑click), delete key to remove; drag to timeline; truncates long names (full name on hover).
  • Tracks: per‑track Mute / Solo / Lock and color labels; header on the left stays aligned with the ruler.
  • Trim & move: draggable left/right handles with snap grid (adjustable step and tolerance); ghost preview shows where a clip will land before drop.
  • Ripple / Glue: ripple delete closes gaps; glue snapping keeps adjacent clips tight.
  • Keyboard: ←/→ = ±1 frame (Shift = ±10), ↑/↓ move to prev/next compatible track, Delete remove, Ctrl+Delete ripple delete, Ctrl+G group, Ctrl+Shift+G ungroup.
  • Markers & nav: M add marker; , / . go to prev/next; Start/End buttons jump to project or selection edges.
  • Zoom & scroll: mouse‑wheel timeline zoom, zoom to selection / fit, horizontal scrollbar.
  • Preview: resizable popup for images/video/audio with play/pause & scrub; stops playback on close.
  • Layout: draggable divider between media panel and timeline to allocate space.

RIFE Interpolation

Out‑of‑the‑box • Queue‑native • Superfast mode
  • Zero‑setup: models ship in assets and auto‑extract to models/rife/default on first run. Choices persist in QSettings.
  • Device auto‑pick: GPU if available, otherwise CPU fallback. No prompts.
  • One‑click run: default target = 2× input FPS, output to outputs/interpolate/…, overwrite off, atomic writes, per‑job logs.
  • Queue integration: enqueues as rife job with ETA + progress; cancel cleans temp; “Open log” on error.
  • Slow‑mo: multiplier from 0.15× → 1.0× creates smooth slow‑motion with proper audio retiming.
  • Interpolation: multiplier 1.0× → 4.0× (2× default) generates true in‑between frames.
  • Profiles: 1–5 slider from minimal‑load to high‑quality; defaults are safe and responsive.
  • Superfast (NCNN/Vulkan): optional toggle to run rife-ncnn-vulkan directly (blocks UI for maximum speed). Uses bundled models (v4.6/HD/UHD/Anime).
  • Player workflow: auto‑play in internal player on finish, plus a “result in external player” button.
  • Advanced: streaming (low‑memory) path, planned model selector with auto‑install, per‑profile CRF/preset defaults.

Tools — full list

  • Speed: slow down or speed up; enqueues jobs; batch for many files.
  • Resize: set width/height (keeps aspect if desired). Save/load presets.
  • Export GIF: 2‑pass palette (palettegen → paletteuse) with dithering for high quality.
  • Extract frames: export frames from a video to JPG/PNG for quick picking or analysis.
  • Reverse Video: fixes “backwards” AI videos; optional boomerang mode for loops.
  • Compare: load two media files and slide/zoom to compare (great for before/after).
  • Multi‑Rename: bulk rename with prefix/suffix, numbering, and find/replace.
  • Background Remove/Replace: remove backgrounds or swap them with a prompt.
  • Trim: cut a time range by start/end (flexible HH:MM:SS.mmm). Generate preview builds a thumbnail grid (6–60) with a Size slider (24–140px). Left‑click a thumb sets Start, right‑click sets End. A draggable range bar mirrors Start/End and dims thumbnails outside the kept region. Manual generation (no auto‑regen) keeps things responsive. Batch and queue supported; modes: Fast copy (keyframe) or Precise re‑encode.
  • Crop: crop a region (e.g., top/bottom band removal) and save as a new video.
  • Quality / Size Video: choose CRF (quality) or Bitrate; presets ultrafast → veryslow; codecs H.264/H.265/AV1 (CPU) and NVENC (GPU); audio copy/AAC/Opus; containers MP4/MKV/MOV.
  • Image Quality / Convert: convert to JPG/WEBP/PNG with quality slider (1–100), optional max width/height, and EXIF keep/remove. Target size mode (KB/MB) available.
  • Add Audio: mux/replace an external audio track onto a video.
  • Batch tools: quick prompts for multi‑file/folder runs across Speed, Resize, GIF, Trim, Crop, Quality, and Image Convert.

Queue — full feature list

Advanced, resilient processing queue with rich job rows and player integration.

  • Safe delete/cleanup with guarded file ops.
  • Theme‑aware icons and compact, hoverable row cards.
  • Never returns the source file by mistake.
  • Robust output resolver: single file, lists, or dir with scoring.
  • Reorder, delete, pause/resume, recover running → pending.
  • Groups for Running/Pending/Finished; compact rows with status strip and progress.
  • ETA calculation with small fudge to avoid early zero.
  • Context menu and toolbuttons: Play, Open folder, Delete, View JSON.
  • Auto‑play latest finished output (config‑persisted toggle).
  • Deterministic output resolver with name/time/size scoring.
  • Internal player preferred; OS fallback; Recents list update.
  • Theme‑aware icons, hover effects, and live updates while running.
  • Paranoid file safety: refuses to delete source; errors handled gracefully.
